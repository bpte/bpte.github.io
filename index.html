sprintf函数

sprintf函数

C语言头文件： <stdio.h>

用法：int sprintf(char *str, const char*format,... )

功能：将格式化的数据写入某个字符串缓冲区。

返回值：字符串长度

sprintf函数与printf函数的区别:
sprintf的作用是将一个格式化的字符串输出到一个目的字符串中，而printf的作用是将一个格式化的字符串输出到屏幕。sprintf的第一个参数应该是目的字符串，如果不指定这个参数，执行过程中出现”该程序产生非法操作，即将被关闭……”的提示。

sprintf常见应用：
sprintf(s, "%d", 123); //把整数123打印成一个字符串保存到s中

Note：
C语言对数组操作时不检查数组的长度，如果str的长度不够，sprintf()很容易造成缓冲溢出，带来意想不到的后果，黑客们常常用这个弱点去攻击看上去安全的系统。

strncpy函数

strncpy函数

C语言头文件： <string.h>

用法： char* strnpy(char *dest, const char* src，int n);

功能：把src指向的字符串复制到dest,最多复制n个字符，当src的的长度小于n时，dest将剩余部分用空字节填补。

Note:

1.当src字符串长度小于n时，在拷贝完字符串后，剩余部分用空字节填补，直到n个，strnpy不会向dest追加’\0’。
2.src和dest区域不能重叠，且dest必须有足够的空间放置n个字节。

strncpy函数与strcpy函数的区别

1.strcpy就是用来复制字符串的，而strncpy函数用来复制字符串的前n个字符。
2.不像strcpy函数，strncpy函数不会向dest追加’\0’。

strcpy函数

strcpy函数

C语言头文件: <string.h>

用法： char *strcpy(char *destination, const char *source);

功能：strcpy函数将含有’\0’结束符的字符串复制到另一个地址空间

返回值：返回的是目标字符串的首地址

Notes：
1.目标空间必须足够大，以确保能够放得下复制过来的字符串。
2.目标空间必须可变，即 char *destination不能用const修饰，并且目的地空间的字符串不能时常量字符串，常量字符串不能被修改。
3.源字符串必须含有’\0’。
4.strcpy函数会将源字符串的’\0’也拷贝到目的地空间

返回值：返回的时目的字符串的首地址。

strcmp函数

strcmp函数

C语言头文件：<string.h>
C++头文件：

用法：int strcmp( const chars1, const chars2);

功能：用来比较字符串的大小。
比较规则：
若s1>s2,返回为正数。
若s1=s2,返回为0。
若s1<s2,返回为负数。
即两个字符串自左向右逐个字符按照ASCLL码进行比较，直到出现不同字符或者出现\0为止。

返回值；返回s1字符与s2字符的差值。

实现原理：
首先strcmp()先将s1的第一个字符减去s2第二个字符，如果两者的差值为0，就继续比较s1和s2的第二个字符，直到出现\0或者差值不为零为止，如果差值不为零，就将差值返回。例如字符”Ac”和”比较则会返回字符”A”(65)和字符”b”(98)的差值-33。

注意事项：
1.联合体类型与结构体类型的定义方法很类似，只是将关键字由struct换成了union。但是他们在内存分配上却有着本质的区别，结构体变量所占用的内存长度是所有元素 所占用的内存总和，而联合体变量所占用的内存长度是其中最长元素的长度。

三目运算符

三目运算符是唯一一个有3个操作数的运算符，有时又称为三元运算符，一般用于赋值语句

形式：条件表达式1？表达式2;表达式3

规则：如果表达式1为真，就执行表达式2并返回表达式2的结果；如果表达式1为假，则执行表达式3并返回表达式3的结果

例如：
MAX = (a > b) ? a : b
如果a>b,MAX=a;
如果a<b,MAX=b;
