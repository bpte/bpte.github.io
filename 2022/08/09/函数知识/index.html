补充函数知识

#rand()函数
在C语言中，我们一般使用<stdlib.h>下的rand（）函数来生成随机数，他的用法为：
int rand(void)

下面是一个生成随机数的例子
#include <stdio.h>
#include <stdlib.h>

int main(void)
{
int a = rand();
printf(“%d\n”,a);
return 0;
}

###随机数的本质
多次运行以上的代码，你会发现得到的值是一致的。实际上，rand()函数产生的随机数都
是伪随机数，是根据一个数值按照某一个公式推导出来的，这个数值被我们称为“种子”。

种子每次启动时都是随机的，但是计算机一旦启动后这个值就不会再变了化了，也就是说，
每次启动计算机后，种子的值都是固定的了，所以根据公式推出来的结果也是固定的了。

###播种
我们可以通过srand()函数进行重新“播种”，这样种子就会变化。srand()的用法为：
void srand(unsighed int seed)

它需要一个unsighed int类型的参数。通常情况下，我们使用时间作为参数，只要每次播种的时间不同，播种出来的结果(即随机数)自然也不相同。

使用<time.h>头文件里的time()可以得到目前的时间(精确到秒)，就像下面这样：
srand((unsighed)time(NULL))

###生成一定范围的随机数

在实际的应用中，我们经常采用取模的方式生成一定范围的随机数
例如：
rand()%5; //代表此时生成的随机数的范围是0~4，一旦大于5，就会被5整除

rand()%10 + 15; //产生15~24的随机数

###连续生成随机数
有时候我们需要连续生成多个随机数，最容易想到的方法就是利用循环
方法一：
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(void)
{
int a;
for(int i=0;i<5;i++)
{
srand((unsighed)time(NULL));
a = rand();
printf(“%d “,a);
}
return 0;
}
输出：132009022 132009022 132009022 132009022 132009022
分析：这里没有出现连续随机数的原因是for循环运行的速度太快了，连一秒的时间都不到就运行完了，而time()函数是指精确到秒的，所以所有的种子都一样。如果需要生成连续的随机数，把srand((unsighed)time(NULL))放到for循环的外面就可以了，如方法二所示：

方法二：
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(void)
{
int a;
srand((unsighed)time(NULL));
for(int i=0;i<100;i++)
{
a = rand();
printf(“%d “,a);
}
return 0;
}
输出：664229233 496172995 2049999013 713627486 767289124
